			 Calculadora de IMC – Android (Kotlin)

Aplicativo Android nativo desenvolvido em Kotlin para cálculo do Índice de Massa Corporal (IMC),
com layout limpo, validação de dados de entrada e classificação automática conforme os padrões da OMS.

O projeto foi construído com foco em boas práticas, clareza de código e experiência do usuário.


- Objetivo do Projeto

Criar um aplicativo simples, funcional e bem estruturado para:

* Praticar desenvolvimento Android nativo

* Trabalhar validação de entrada de dados

* Aplicar lógica de negócio de forma clara

* Desenvolver interfaces responsivas com ConstraintLayout


- Funcionalidades

* Entrada de peso (kg) e altura (cm)

* Validação para impedir:

	. Campos vazios

	. Valores inválidos ou zero

* Conversão automática de centímetros para metros

* Cálculo do IMC

* Classificação automática do resultado:

	. Abaixo do peso

	. Peso normal

	. Sobrepeso

	. Obesidade

* Exibição do resultado formatado com duas casas decimais


- Tecnologias Utilizadas

* Kotlin

* Android Studio

* ConstraintLayout

* Componentes nativos do Android:

	. TextView

	. EditText

	. Button


- Layout da Tela

O layout foi desenvolvido utilizando ConstraintLayout, garantindo:

* Responsividade para diferentes tamanhos de tela

* Organização visual

* Código limpo e de fácil manutenção
